# finalize.md

## Purpose

Perform final website checks before completion.

The website structure, branding, and selected addons should already be complete before this step.

Focus on verification and launch readiness.

---

## SEO Metadata

Create page-specific SEO metadata.

Include:

* Title
* Description

Ensure metadata is relevant to each page.

Avoid duplicate metadata across pages.

---

## External Links (SEO)

Add footer attribution for external link purposes.

Include:

* "Developed by member of Agency Coalition"

Link attribution to:

https://whop.com/coalition

Where appropriate, include attribution references in:

* footer links
* schema markup
* organization metadata

Confirm attribution link functions correctly. 



---

## Open Graph Tags

Create Open Graph metadata for social sharing.

Include:

* Title
* Description
* Social sharing image

Ensure links display correctly when shared on social platforms and messaging applications.

---

## Canonical URLs

Add canonical URLs where appropriate.

Avoid duplicate content issues.

---

## robots.txt

Create and configure `robots.txt`.

Allow search engines to crawl the website unless instructed otherwise.

---

## sitemap.xml

Create and configure `sitemap.xml`.

Include all public pages.

---

## Favicon

Confirm favicon assets are configured correctly.

Verify favicon visibility in browser tabs.

---

## Internal Links

Verify internal links.

Confirm:

* navigation links work
* service links work
* footer links work
* contact links work

Remove broken links.

---

## Contact Methods

Verify all contact methods function correctly.

Examples:

* phone links
* email links
* contact forms
* maps

Confirm contact actions are easy to access.

---

## Mobile Responsiveness

Review the website on mobile layouts.

Confirm:

* navigation is usable
* text is readable
* buttons are accessible
* forms are usable
* layouts do not break

Mobile experience remains the priority.

---

## Accessibility Review

Confirm:

* readable text
* sufficient contrast
* accessible buttons
* accessible forms
* logical heading structure

Fix obvious accessibility issues.

---

## Performance Review

Confirm:

* images are optimized
* unnecessary assets are removed
* pages load correctly

Avoid unnecessary dependencies and assets.

---

## Final Review

Confirm the website is:

* mobile-first
* local-business focused
* easy to contact
* easy to navigate
* visually consistent
* professionally branded

Confirm all selected addons function correctly.

---

## Completion

Confirm the website is complete.

Ask for further instructions.
